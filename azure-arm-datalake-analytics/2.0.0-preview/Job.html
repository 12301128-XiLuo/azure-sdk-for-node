build(accountName, parameters, optionsopt, optionalCallbackopt) → {function|Promise}

Builds (compiles) the specified job in the specified Data Lake Analytics
account for job correctness and validation.

Parameters:
Name Type Attributes Description
accountName string

The Azure Data Lake Analytics account to execute
job operations on.
parameters object

The parameters to build a job.

Properties
Name Type Attributes Description
name string

the friendly name of the job.
type string

the job type of the current job (Hive or
USql). Possible values include: 'USql', 'Hive'
degreeOfParallelism number <optional>

the degree of parallelism
used for this job. This must be greater than 0, if set to less than 0 it
will default to 1.
priority number <optional>

the priority value for the current
job. Lower numbers have a higher priority. By default, a job has a priority
of 1000. This must be greater than 0.
logFilePatterns array <optional>

the list of log file name
patterns to find in the logFolder. '' is the only matching character
allowed. Example format: jobExecution.log or mylog.txt
properties object

the job specific properties.

Name Type Attributes Description

run.

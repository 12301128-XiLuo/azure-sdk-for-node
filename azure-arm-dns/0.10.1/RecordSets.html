createOrUpdate(resourceGroupName, zoneName, relativeRecordSetName, recordType, parameters, optionsopt, optionalCallbackopt) → {function|Promise}

Creates or updates a record set within a DNS zone.

Parameters:
Name Type Attributes Description
resourceGroupName string

The name of the resource group.
zoneName string

The name of the DNS zone (without a terminating
dot).
relativeRecordSetName string

The name of the record set, relative
to the name of the zone.
recordType string

The type of DNS record in this record set. Record
sets of type SOA can be updated but not created (they are created when the
DNS zone is created). Possible values include: 'A', 'AAAA', 'CNAME', 'MX',
'NS', 'PTR', 'SOA', 'SRV', 'TXT'
parameters object

Parameters supplied to the CreateOrUpdate
operation.

The etag of the record set.
metadata object <optional>

The metadata attached to the record
set.
tTL number <optional>

The TTL (time-to-live) of the records in
the record set.
aRecords array <optional>

The list of A records in the record
set.
aaaaRecords array <optional>

The list of AAAA records in the
record set.
mxRecords array <optional>

The list of MX records in the record
set.
nsRecords array <optional>

The list of NS records in the record

The list of PTR records in the record

The list of SRV records in the record

The list of TXT records in the record

cnameRecord object <optional>

The CNAME record in the record
set.

Properties
Name Type Attributes Description
cname string <optional>

The canonical name for this
CNAME record.
soaRecord object <optional>

The SOA record in the record set.

Properties
Name Type Attributes Description

The domain name of the

duration.
options object <optional>

Optional Parameters.

Properties
Name Type Attributes Description

The etag of the record set. Omit this

will be ignored.
